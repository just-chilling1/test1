# Product Skeleton

White-label Next.js + Supabase starter. Auth, shell, and promos are built-in; features plug in via config.

## Developer: read this

**[DEVELOPER-SETUP.md](./DEVELOPER-SETUP.md)** — complete step-by-step guide (local setup → features → deploy → launch).

## Quick start

```bash
cp .env.local.example .env.local
npm install
npm run dev
```

Then follow **DEVELOPER-SETUP.md** from Step 2 onward.
